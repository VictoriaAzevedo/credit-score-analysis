# Credit Score Analysis

Projeto desenvolvido como parte do desafio final do **Bootcamp de Analista de Dados da Re/Start**.

## Objetivo

Analisar o perfil financeiro dos clientes e identificar fatores associados à classificação do score de crédito.

## Ferramentas utilizadas

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Power BI

## Dataset

Credit Score Classification (Kaggle) - https://www.kaggle.com/datasets/parisrohan/credit-score-classification 

## Etapas do projeto

- Exploração dos dados
- Limpeza e preparação
- Análise exploratória (EDA)
- Construção do dashboard
- Geração de insights
